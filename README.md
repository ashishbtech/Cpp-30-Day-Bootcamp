# 🚀 C++ 30-Day Bootcamp: The DSA Training Arc

Welcome to my 30-Day C++ Bootcamp repository. 

As a Computer Science student with a foundational understanding of C, this repository serves as a structured bridge to mastering **modern C++**. The goal of this 30-day sprint is not just to learn a new syntax, but to forge a high-speed, highly optimized toolkit for tackling complex Data Structures and Algorithms (DSA) and preparing for technical placement interviews.

### 🎯 Core Objectives
* **Phase 1:** Unlearn the limitations of pure C and embrace modern C++ mechanics (References, Dynamic Memory, Range-based loops).
* **Phase 2:** Master Object-Oriented Programming (Classes, Inheritance, Polymorphism) to write clean, modular code.
* **Phase 3 & 4:** Weaponize the Standard Template Library (STL). Moving beyond raw arrays to leverage `vectors`, `maps`, `sets`, and built-in algorithms like `std::sort` and `lower_bound` for $O(1)$ lookups and $O(N \log N)$ optimizations.

All code is written, tested, and categorized daily.
